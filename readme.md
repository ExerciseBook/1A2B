# 1A2B UWP版本

## 概述
- 一个普通的大作业
- 一个普通的小游戏
- 一个普通的新人作


## 游戏规则
- 在游戏开始时，生成各个数为互不相同且没有前导零的一个四位数。
- 玩家有10次机会去猜测这个数字是什么。
- 玩家每一次尝试，评判系统会反馈给玩家，你有**多少个数字数码猜对了并且位置也猜对了**和**多少个数字数码猜对了但是位置没猜对**。
- 请发挥你的聪明才智来解决这个问题吧~
